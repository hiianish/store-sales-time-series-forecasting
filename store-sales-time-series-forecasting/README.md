# 📊 Store Sales Time Series Forecasting

An end-to-end machine learning project for forecasting store sales using time-series analysis and providing business insights through an interactive dashboard.

---

##  Features

- Time-series forecasting using ML models  
- Feature engineering (lag features, rolling statistics)  
- Sales trend and seasonality analysis  
- Holiday and promotion impact insights  
- Product category filtering  
- Interactive Streamlit dashboard  

---

##  Tech Stack

**Machine Learning & Forecasting**
- Scikit-learn
- XGBoost

**Data Analysis**
- Pandas
- NumPy

**Visualization**
- Matplotlib
- Seaborn

**Frontend / Dashboard**
- Streamlit
- HTML/CSS

**Tools**
- Python
- Git & GitHub
